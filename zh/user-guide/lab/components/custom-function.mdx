**节点说明**：`自定义函数`节点可以指定输入参数和函数内容（目前支持 <Tooltip tip="Groovy 是一门运行在 JVM 上的动态编程语言，与 Java 无缝兼容，但代码更简洁、灵活。">Groovy</Tooltip> 语言），您可以按照自己的需要，对数据进行更加细致的处理，得到函数处理完成后的结果进行输出。

![](/images/components/custom_function_1.png "")

## 配置

定义输入变量，可以是上个节点的参数应用、也可以自定义参数，然后在代码中引用这些变量。经过指定的函数运算，得到理想输出值。输出结果可以通过返回内容是否开启来决定是用来内部计算还是展示给用户。

![](/images/components/custom_function_2.png "")

<Warning>如果您的函数内容有误，会直接抛出异常。</Warning>
