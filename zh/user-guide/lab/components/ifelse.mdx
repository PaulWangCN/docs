**节点说明**：判断器组件通过根据您定义的条件将执行路由到不同路径，为您的工作流添加决策逻辑。它评估变量并确定您的工作流应该遵循哪个分支。
![](/images/components/ifelse_1.png "")

## 分支逻辑
该节点支持多个分支路径来处理复杂的决策树：
- `IF` 路径在主要条件评估为真时执行。
- `ELSE IF` 路径在 `IF` 条件为假时提供按顺序检查的附加条件。您可以添加多个 `ELSE IF` 分支来处理复杂逻辑。
- `ELSE` 路径作为没有条件匹配时的后备选项，确保您的工作流始终有路径可循。

## 条件类型
- **为空 / 不为空**：判断变量值是否为空。
- **包含 / 不包含**：判断变量值是否包含指定内容。
- **等于 / 大于等于 / 大于 / 小于等于 / 小于**：判断变量值是否满足指定比较。
- **长度等于 / 长度大于等于 / 长度大于 / 长度小于等于 / 长度小于**：判断变量值长度是否满足指定比较。
- **为真 / 不为真**：判断变量值是否为真。
