**节点说明**：变量赋值可以修改流程中的变量值。

支持的修改操作包括：
- 覆盖：将新的值覆盖到原有值上。
- 追加：在原有值上进行值的追加，如列表增加新的元素。
- 清空：将变量值设置为空。

赋值来源包括：
- 自定义：赋值为指定的值。
- 引用变量：赋值为已存在的变量值。
![](/images/components/var_assignment_1.png "")

## 多变量赋值
若需要在`变量赋值器`节点同时设置多个变量，点击添加按钮，可以同时对多个变量进行赋值。
