**节点说明**：`开始节点`有且唯一，不能删除和复制，定义了整个流程上下文的全局变量，方便后续节点的取用。

![](/images/components/start_node_1.png "")

## 全局变量

全局变量贯穿整个流程的上下文，可在任何节点中获取使用。目前内置定义了如下全局变量：

- **当前时间 `{time}`**：当前对话的时间。
- **历史聊天记录 `{history_context}`**：对话的历史记录。
- **应用 ID `{application_id}`**：应用ID。
- **对话 ID `{chat_id}`**：对话标识。
- **对话记录 ID `{chat_record_id}`**：对话记录ID。
- **用户 ID `{user_id}`**：用户ID。

## 输出参数

输出参数默认**用户问题 `{question}`**，若您在**基础信息**节点中打开相应功能的开关，可以额外得到相关的输出参数。

- **用户问题 `{question}`**：客户端输入的提问信息。
- **文档 `{document}`**：客户端上传的文档，需要在**基础信息**节点中打开**文件上传**功能。
- **联网搜索 `{use_network_search}`**: 联网搜索查询的结果，需要在**基础信息**节点中打开**联网**功能。
- **深度思考 `{use_think}`**: 深度思考的结果，需要在**基础信息**节点中打开**深度思考**功能。
- **本地知识库 `{use_search_dataset}`**: 查询知识库的结果，需要在**基础信息**节点中打开**本地知识库**功能。
- **记忆体 `{use_long_memory}`**: 通过记忆力查询的结果，需要在**基础信息**节点中打开**记忆体**功能。
