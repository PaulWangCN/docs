**节点说明**：`文本转语音` 节点可以将您输入的文本，转化为自然、流畅的语音音频。这对于创建语音助手、有声内容、播客或为应用增加语音交互能力非常有用。

## 核心功能

- **自然语音合成**：将输入的文字内容转换为接近真人发音的语音。
- **多参数调控**：支持调节语速、音调、音量等，定制个性化的语音效果。
- **多格式输出**：生成的音频支持 `MP3`、`WAV` 等多种常见格式。

## 参数配置

配置 `文本转语音` 节点时，您需要设置以下参数：

| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| **输入文本** | `String` | 是 | 需要转换为语音的纯文本内容。支持中文、英文等多种语言。 |
| **语音模型/发音人** | `Enum` | 是 | 选择合成语音所使用的声学模型或发音人角色。不同选项代表不同的音色（如男声、女声、儿童声等）和语言风格。 |
| **语速** | `Number` | 否 | 控制语音播放的速度。通常为 `0.5`（半速）到 `2.0`（倍速）之间的值，`1.0` 为正常语速。 |
| **音调** | `Number` | 否 | 控制语音的音高。调整此值可以改变声音的低沉或尖锐程度。 |
| **音量** | `Number` | 否 | 控制输出音频的音量大小。 |
| **输出格式** | `Enum` | 否 | 选择生成的音频文件格式，如 `MP3`、`WAV` 等。默认为 `MP3`。 |

## 使用方法

![](/images/components/text_to_speech_1.png "")

1.  **添加节点**：在工作流画布中，从节点库中找到 **`文本转语音`** 节点，将其拖拽到画布上。
2.  **连接输入**：将上一个节点的输出（通常是包含待转换文本的变量，如 `{{某LLM节点.answer}}` 或 `{{查询结果.text}}`）连接到本节点的 **`输入文本`** 端口。
3.  **配置参数**：在节点的属性面板中，按上表说明配置各项参数。最关键的步骤是选择一个合适的**语音模型/发音人**。
4.  **连接输出**：将本节点的 **`音频数据`** 或 **`音频URL`** 输出端口连接到后续节点。例如，可以连接到一个**指定回复**节点用于播放，或连接到一个**文件存储**节点进行保存。

## 节点输出

`文本转语音` 节点成功执行后，会提供以下输出变量，供后续节点使用：

| 输出变量名 | 类型 | 说明 |
| :--- | :--- | :--- |
| `audio_data` | `File` (Base64) | 生成的音频文件的二进制数据，以 Base64 字符串格式编码。适用于需要直接嵌入或处理的场景。 |
| `audio_url` | `String` (URL) | 生成的音频文件在服务器上的临时存储地址。适用于需要前端播放或链接分享的场景。**请注意，此链接可能有过期时间。** |

## 应用示例

**场景：智能语音天气播报**
1.  一个 **`LLM`** 节点根据用户查询生成一段文本回复：“北京今天晴，气温 15 到 25 摄氏度，空气质量良。”
2.  将此文本（`{{LLM节点.answer}}`）作为 **`文本转语音`** 节点的输入。
3.  在`文本转语音`节点中，选择一位发音清晰、亲切的**女声**发音人，语速设为 `1.1`。
4.  节点输出音频文件，并连接到一个 **`指定回复`** 节点，以语音卡片的形式将天气情况播报给用户。

## 注意事项

- **发音人选择**：不同发音人对中文、英文或混合文本的支持效果可能不同，请根据文本主要语言进行选择测试。
- **文本长度**：过长的文本可能会导致合成时间较长。对于极长的文本，可考虑分段处理。
- **网络依赖**：语音合成通常需要调用云端服务，请确保网络连接稳定。
- **费用说明**：使用此节点可能会产生额外的语音合成服务费用，具体请参考平台计费说明。

通过 `文本转语音` 节点，您可以轻松地为您的智能体增添“声音”，创造更丰富、更人性化的交互体验。
