清奇领航通过 `MCP` 管理和 `MCP` 广场提供了全周期 `MCP` 服务，既支持在平台内部（如**智能体**、**工作流**）进行配置，也支持通过外部调用集成至第三方应用或个人项目。

- MCP 管理
团队空间下的 `MCP` 管理能力，为管理员账号提供 `MCP` 注册与管理中心，平台支持 `SSE` 和 `STDIO` 两种协议 `MCP` 服务，支持服务协议元数据定义、版本控制及生命周期管理。并提供服务工具调用日志监控能力。

- MCP 广场
1. 以卡片形式展示所有可用的 `MCP Server`，包含名称、`Logo`、简短描述等关键信息。
2. 部署 `MCP` 服务器，为用户生成平台专属 `SSE` 连接 `URL`，用于智能体或工作流中接入 `MCP`。
3. 集成式服务调试工具，在图形化界面中模拟输入参数，实时发起调用并查看结果，使服务调试与验证工作变得简单高效，降低了开发与运维门槛。
