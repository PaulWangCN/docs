## MCP 管理
前往清奇领航后台系统，登录管理员账号，对 `MCP` 服务进行管理，包含 `MCP` 服务注册、`MCP` 版本管理、`MCP` 生命周期管理。

### 服务注册

1. **创建服务分类**

选择 **MCP 管理** > **分类管理** > **新增分类**，点击新增分类，输入分类信息，点击确定保存创建分类，创建后的分类在创建服务注册时可选择使用。
![](/images/mcp/mcp_1.png "")

2. **创建服务注册**

选择 **MCP 管理** > **服务管理** > **新增服务**，点击新增服务，输入服务信息：基础信息、配置信息、服务介绍，点击保存按钮创建并注册服务。创建成功的服务默认启用、发布，并同步到 MCP 广场提供使用。
![](/images/mcp/mcp_2.png "")

### 版本管理

1. **版本新增发布**

选择 **MCP 管理** > **服务管理**，在服务列表中选择需要新增或发布版本的服务，操作栏中点击编辑按钮，编辑服务版本号信息，新版本号已存在则编辑对应版本服务信息，不存在则新增创建对应版本服务。

点击保存按钮则预置版本不发布，点击发布按钮则新增并发布该版本，并把原发布版本置为未发布。

![](/images/mcp/mcp_3.png "")

![](/images/mcp/mcp_4.png "")

2. **历史版本管理**

选择 **MCP 管理** > **服务管理**，在服务列表中选择需要管理历史版本的服务，操作栏中点击编辑按钮进入服务编辑页，在版本列表中选择需要管理的版本，对版本进行编辑或发布。

![](/images/mcp/mcp_5.png "")

### 生命周期
1. **启用、禁用、删除服务**

选择 **MCP 管理** > **服务管理**，在服务列表中选择需要管理生命周期的服务，操作栏中点击对应操作按钮：禁用服务可操作启用、删除，启用服务可操作禁用、删除。
![](/images/mcp/mcp_6.png "")
MCP 生命周期管理同步影响 MCP 广场服务列表，禁用或删除的服务在 MCP 广场不展示。

## MCP 广场
前往清奇领航中台系统，登录用户账号，进入 MCP 广场，应用管理员在清奇领航后台系统维护的 MCP 服务：部署连接 MCP 服务、调试验证 MCP 服务、连接使用 MCP 服务。

- **搜索选择 MCP 服务**

选择 **MCP 广场** > 根据服务分类或名称搜索所需 **MCP 服务**：
![](/images/mcp/mcp_7.png "")

- **部署连接 MCP 服务**

在 **MCP 广场**选择所需 **MCP 服务**，进入服务详情页，在获取 `MCP` 服务器处选择 `SSE` 模式，输入服务所需环境变量，点击连接，为用户生成 MCP 服务专属 `SSE URL`。
![](/images/mcp/mcp_8.png "")
![](/images/mcp/mcp_9.png "")

- **调试验证 MCP 服务**

部署连接 MCP 服务后，进入工具调试，验证部署连接的服务工具
![](/images/mcp/mcp_10.png "")

- **连接使用 MCP 服务**
