## 管理知识中心
> 知识库管理页仅面向团队所有者、团队管理员、拥有编辑权限的角色开放。

点击平台顶部的“知识中心”按钮，选择需要管理的知识库。轻点左侧导航中的设置进行调整。您可以在此处调整知识库名称、向量模型和描述。
- **知识库名称**：用于区分不同的知识库。
- **向量模型**：用户嵌入分段内容，生成向量
- **知识库描述**：用于描述知识库内文档代表的信息。
![](/images/kb/kb_manage_1.png "")

## 维护知识中心内文档
**添加文档**
知识库是文档的集合。文档支持本地上传。知识库内的文档对应数据源中的一个文件单位。点击**知识中心** → **选择知识库** → **上传文件**，在已创建的知识库内上传新的文档。
![](/images/kb/kb_manage_2.png "")

**启用 / 禁用 / 删除文档**

**启用**：处于正常使用状态的文档，支持编辑内容与被知识库检索。对于已被禁用的文档，允许重新启用。已归档的文档需撤销归档状态后才能重新启用。

**禁用**：对于不希望在使用 AI 应用时被检索的文档，可以关闭文档右侧的蓝色开关按钮以禁用文档。禁用文档后，仍然可以编辑当前内容。

**删除**：**⚠️ 危险操作**。对于一些错误文档或明显有歧义的内容，可以点击文档右侧菜单按钮中的删除。

<Danger>删除后的内容将无法被找回，请进行谨慎操作。</Danger>

> 以上选项均支持选中多个文档后批量操作。
![](/images/kb/kb_manage_3.png "")

**管理文本分段**
查看文本分段知识库内已上传的每个文档都会以文本分段（`Chunks`）形式进行存储。点击文档标题，在详情页中查看当前文档的分段列表，每页默认展示 `30` 个区块，滚动加载。

**内容分为父分段和子分段。**
- **父分段**选择知识库内的文档后，您将会首先看到父分段的内容。父分段提供更加完整的上下文信息。
- **子分段**一般为段落中的某个句子（较小的文本块），包含细节信息。各个分块均会展示字符数以及被检索召回的次数。

<Note>检查分段质量</Note>

文档分段对于知识库应用的问答效果有明显影响，在将知识库与应用关联之前，建议人工检查分段质量。检查分段质量时，一般需要关注以下几种情况：
- **过短的文本分段**，导致语义缺失；
- **过长的文本分段**，导致语义噪音影响匹配准确性；
- **明显的语义截断**，在使用最大分段长度限制时会出现强制性的语义截断，导致召回时缺失内容；

**添加文本分段**

知识库中的文档支持单独添加文本分段。点击分段列表顶部的**添加分段**按钮，可以在文档内自行添加一个自定义父分段。
![](/images/kb/kb_manage_4.png "")

**编辑文本分段**

父分段包含其本身所包含的子分段内容，两者相互独立。您可以单独修改父分段或子分段的内容。

修改父分段后，点击 **保存** 后将不会影响子分段的内容。如需重新生成子分段内容，轻点 **保存并重新生成子分段**。
![](/images/kb/kb_manage_5.png "")

**修改已上传文档的文本分段**

已创建的知识库支持重新配置文档分段。

您可以访问**分段设置**，点击**保存**按钮以保存对分段设置的修改，并重新触发当前文档的分段流程。当您保存设置并完成嵌入处理后，文档的分段列表将自动更新。
![](/images/kb/kb_manage_6.png "")
